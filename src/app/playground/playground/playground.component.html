Pipes

Pipes são um operador especial em expressões de template Angular que permite que você transforme dados declarativamente em seu template. Pipes permitem que você declare uma função de transformação uma vez e então use essa transformação em múltiplos templates. Pipes Angulares usam o caractere de barra vertical ( |), inspirado no pipe Unix.

Para mais informações, acesse a documentação oficial do Angular sobre Pipes.

NgRx no Angular: Gerenciamento de Estado

O NgRx é uma biblioteca para gerenciamento de estado no Angular, baseada no padrão Redux. Ele permite gerenciar o estado global da aplicação de maneira previsível, escalável e reativa usando RxJS.

Para mais informações, acesse a documentação oficial do NGRX.

Angular Material

Angular Material é uma biblioteca de componentes UI (Interface de Usuário) que segue as diretrizes de design do Material Design do Google. Ela fornece uma série de componentes prontos para uso, como botões, caixas de diálogo, formulários, tabelas, menus, entre outros,

Para mais informações, acesse a documentação oficial do Angular Material.
